Features

  • Displays load icon while fetching data
  • Color theme can be changed. Current one is hard coded. However, it should ideally be getting picked from design guide
  • All the text on graph - title, axis names, series names can be translated
  • Tooltip displayed on hover of each column
  • On window resize graph is resized accordingly

Note: The colour theme, change theme and refresh button are demonstration purpose only. The latency is added while getting data.
